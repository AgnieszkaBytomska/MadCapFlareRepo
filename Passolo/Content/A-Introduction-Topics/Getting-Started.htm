Getting Started

Document is an outcome of research, discussions and Proof of Concept for PBI # 48139 [MRD] PoC Passolo: Prepare solution which supports import/export data (translations) from Passolo tool to something which can be easily leveraged by MO. The aim is to have an user friendly tool which will improve better quality of translated application, reduce cost and time of localization process.

This document is intended for translators who will translate strings for software through SDL Passolo Translator Editor 2018,a software localization tool.

Translator will find below information on installing SDL Passolo program and the workflow of translation and review process.
