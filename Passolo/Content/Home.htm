Passolo Translation Workflow Process

The aim of this page is to explain to an user the Passolo tool which will improve better quality of translated application, reduce cost and time of localization process. This document is intended for translators who will translate strings for software through SDL Passolo Translator Editor 2018,a software localization tool. Translator will find below information on installing SDL Passolo program and the workflow of translation and review process.

SDL Passolo is an application that supports you in translating text resources and localizing software. In this context, all text components of a software that are displayed on the screen for the user are designated as text resources.

Frequently Asked Questions

Can I use my SDL Passolo 2018 license to license Trados Studio 2017?

No, the license for Passolo 2018 is unable to also activate Trados Studio 2017. You can upgrade your license and receive a new licensing code.

 Can I install the SDL License Server Manager 2018 on UNIX OS?

No, SDL License Server Manager 2018 does not support UNIX OS.

 Can I split my network license to be used on different servers?

Yes, when activating online, you have to specify the number of activations that you wish to use. Once activated, you can then activate the remaining number of activations on your second server.
